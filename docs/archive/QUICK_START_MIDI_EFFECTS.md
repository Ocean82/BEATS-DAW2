# Quick Start: MIDI Effects

## Getting Started in 3 Steps

### 1. Open MIDI Effects Panel
- Select any MIDI track (synth, drums, or sampler)
- Click **"MIDI FX"** button in the left sidebar
- You'll see 5 effect sections

### 2. Enable an Effect
Each effect has an **ON/OFF** toggle button:
- **Green** = Quantizer is ON
- **Pink** = Chord Generator is ON  
- **Blue** = Note Repeater is ON
- **Yellow** = Arpeggiator is ON
- Transposer is always active (no toggle)

### 3. Play and Adjust
- Play notes on the piano keyboard (bottom of screen)
- Adjust effect parameters with sliders
- Hear changes in real-time!

## Try These Presets

### 🎹 One-Finger Chords
1. Enable **Chord Generator**
2. Set Type: `maj7`
3. Set Voicing: `open`
4. Play single notes → instant chords!

### 🎵 Scale-Locked Improvisation
1. Enable **Quantizer**
2. Set Root: `C`
3. Set Scale: `major`
4. Set Strength: `100%`
5. Play any notes → they snap to C major!

### 🔄 Classic Arpeggio
1. Enable **Arpeggiator**
2. Set Pattern: `up`
3. Set Rate: `1/16`
4. Set Octaves: `2`
5. Hold a chord → automatic arpeggio!

### 🎸 Stutter Effect
1. Enable **Note Repeater**
2. Set Rate: `1/16`
3. Set Repeats: `8`
4. Set Vel Decay: `30%`
5. Play notes → rhythmic stutters!

### 🎼 Transpose Up an Octave
1. Go to **Transpose** section
2. Set Octaves: `+1`
3. All notes play one octave higher!

## Tips

- **Per-Track Effects**: Each track has its own MIDI effects
- **Non-Destructive**: Original MIDI is never changed
- **Real-Time**: All effects work instantly as you play
- **Combine Effects**: Stack multiple effects for complex sounds
- **Experiment**: Try random combinations!

## Keyboard Shortcuts

- **Space**: Play/Pause
- **A-;**: Piano keys (white keys)
- **W,E,T,Y,U,O,P**: Black keys

## Common Questions

**Q: Why can't I see MIDI FX panel?**  
A: Make sure you've selected a MIDI track (not an audio track)

**Q: Effects not working?**  
A: Check that the effect is enabled (toggle should be colored)

**Q: How do I save my settings?**  
A: Settings are saved per-track automatically

**Q: Can I use effects on recorded MIDI?**  
A: Yes! Effects work on both live input and recorded clips
